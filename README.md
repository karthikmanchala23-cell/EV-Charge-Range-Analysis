# Visualization Tool for Electric Vehicle Charge and Range Analysis using MySQL and Tableau

## Project Overview

This project presents a visualization tool for analyzing Electric Vehicle (EV) charge and range data using MySQL and Tableau. The project helps users understand EV performance through interactive dashboards and visualizations.

## Objectives

- Analyze electric vehicle datasets.
- Store and manage data using MySQL.
- Create interactive dashboards using Tableau.
- Compare EV range, efficiency, price, and charging station information.

## Tools Used

- MySQL
- Tableau
- Microsoft Excel

## Dataset

The project uses Electric Vehicle datasets containing information about:

- Electric Cars
- Cheapest EV Cars
- Charging Stations
- EV India Data
