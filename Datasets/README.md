# Datasets

This folder contains all the datasets used in the project.

## Files

- **ElectricCarData_Clean.csv** – Electric vehicle specifications and performance data.
- **Cheapestelectriccars-EVDatabase.csv** – Information about affordable electric vehicles.
- **EVIndia.csv** – Electric vehicle data related to the Indian market.
- **electric_vehicle_charging_station_list.csv** – Charging station details used for analysis.

These datasets were imported into MySQL and used for creating Tableau visualizations.
