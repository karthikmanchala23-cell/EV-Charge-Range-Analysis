# Project Development Phase

## Project Title
Visualization Tool for Electric Vehicle Charge and Range Analysis

## Description
This folder contains the implementation files, datasets, Tableau dashboards, screenshots, and supporting resources used to develop the Electric Vehicle Charge and Range Analysis project.

## Files Included

### 📄 Business Questions
Contains the key business questions that the project aims to answer using data analysis and visualization.

### 📁 Datasets
Includes the CSV files used for analysis, such as electric vehicle information, charging station details, brand data, and price datasets.

### 📁 Tableau
Contains the Tableau workbook and dashboard files developed for visualizing electric vehicle data.

### 📁 Screenshots
Includes screenshots of the dashboards, visualizations, SQL operations, and project implementation for documentation purposes.

## Objective
To implement the complete Electric Vehicle Charge and Range Analysis system by integrating datasets, MySQL, and Tableau to generate meaningful insights through interactive visualizations.
