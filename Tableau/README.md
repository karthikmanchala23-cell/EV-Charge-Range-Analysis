# Tableau Files

This folder contains all Tableau workbook files (.twb) created for the **Visualization Tool for Electric Vehicle Charge and Range Analysis** project.

## Tableau Workbooks

- **Electric Vehicle Charge and Range Analysis.twb** – Final Tableau project workbook.
- **EV Range Analysis.twb** – Analysis of electric vehicle driving range.
- **EV Range Overview.twb** – Overview of EV range distribution.
- **EV Efficiency Analysis.twb** – Analysis of vehicle efficiency.
- **EV Efficiency Overview.twb** – Summary of EV efficiency.
- **Brand Comparison.twb** – Comparison of different EV brands.
- **Brand Range Comparison.twb** – Brand-wise range comparison.
- **EV Brand vs Range.twb** – Visualization of EV brands against driving range.

These Tableau workbooks were used to create interactive dashboards and stories for the project.
