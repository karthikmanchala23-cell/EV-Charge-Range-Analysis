# Doc and Demo

## Project Title
Visualization Tool for Electric Vehicle Charge and Range Analysis

## Description
This folder contains the final project documentation and demonstration materials prepared for project submission and evaluation.

The **EV Analysis Final Report** includes:
- GitHub Repository link
- Tableau Public Dashboard link
- Complete project documentation

## Files Included

### 📄 EV Analysis Final Report
Provides a complete overview of the project, including the problem statement, methodology, implementation, testing, results, conclusion, and the project links (GitHub Repository and Tableau Public Dashboard).

### 🎥 Demonstration Video
Shows the working of the project, including data loading, SQL database operations, Tableau dashboards, and key insights generated from the visualizations.
