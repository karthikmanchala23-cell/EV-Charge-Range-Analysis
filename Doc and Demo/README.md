# Doc and Demo

## Project Title
Visualization Tool for Electric Vehicle Charge and Range Analysis

## Description
This folder contains the final project documentation and demonstration materials prepared for project submission and evaluation.

## Files Included

### 📄 Final Report
Provides a complete overview of the project, including the problem statement, methodology, implementation, testing, results, and conclusion.

### 🎥 Demonstration Video
Shows the working of the project, including data loading, SQL database operations, Tableau dashboards, and key insights generated from the visualizations.

## Objective
To present the complete project documentation and demonstrate the successful implementation of the Electric Vehicle Charge and Range Analysis system.
