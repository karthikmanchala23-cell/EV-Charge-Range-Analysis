# EV Internship Screenshots

This folder contains all screenshots collected during the development and testing of the **Visualization Tool for Electric Vehicle Charge and Range Analysis**.

## Folder Structure

- **MySQL Screenshots** – Database creation, table creation, SQL queries, and query results.
- **Tableau Screenshots** – Worksheets, visualizations, dashboard, and story screenshots.
- **Performance Testing Screenshots** – Screenshots related to project execution and validation.

These screenshots are included as supporting evidence for the internship project.
